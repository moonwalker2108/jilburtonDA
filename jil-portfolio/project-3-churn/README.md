## 🤖 Patient Churn Prediction Model  
Python | Machine Learning | Healthcare Analytics  

Built a machine learning model to identify patients at risk of churn using behavioral and utilization-based features.

---

#### 🔍 Key Highlights
- Cleaned and prepared structured patient data  
- Engineered features related to engagement and utilization  
- Trained classification model to predict patient churn  

---

#### 💡 Business Value
- Identifies high-risk patients early  
- Supports targeted retention strategies  
- Improves patient engagement and continuity of care  

---

#### 🛠 Tools & Techniques
- Python (data analysis and modeling)  
- Pandas (data manipulation)  
- Scikit-learn (machine learning model)
