# Healthcare Plan Churn Model  
**Language:** Python (pandas, scikit-learn)  
**Dataset:** 100k member roster (synthetic)  
**Model:** Random Forest, 87% F1-score  

### Files
- `churn_data_dictionary.csv` – 50-row demo sample  
- `churn_model.py` – train & evaluate script  

### Run
pip install -r requirements.txt  
python churn_model.py
