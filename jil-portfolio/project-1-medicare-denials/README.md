# Medicare Denials Dashboard  
**Tool:** Power BI (DirectQuery)  
**Dataset:** 400 000 Medicare Part-B claims (dummy PHI-stripped)  
**Goal:** Identify top denial reasons & trending revenue loss  

### Key Metrics
- Denied Claim Count & Revenue by Month  
- Top 10 Denial Reason Codes  
- YTD Denial Rate %  

### Publish Steps
1. Open `Medicare_Denials.pbix` → **File → Publish to Power BI**  
2. Check “Allow public sharing” → copy URL → shorten via bitly  
3. Add URL to root README
# 📊 Medicare Claims & Denials Analysis  
**SQL | Tableau | Healthcare Analytics**

Analyzed Medicare Part B claims data to identify denial patterns, revenue loss drivers, and trends over time.

---

## ✨ Impact
- Identified top denial reason codes contributing to lost reimbursement  
- Analyzed monthly denial trends to highlight revenue leakage patterns  
- Calculated denial rate % to evaluate financial performance over time  

---

## 📊 Key Insights
- Denial volume and revenue loss showed consistent monthly trends  
- A small number of denial codes accounted for the majority of losses  
- Denial rate trends revealed opportunities for process improvement
