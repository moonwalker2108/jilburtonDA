# Medicare Denials Dashboard  
**Tool:** Power BI (DirectQuery)  
**Dataset:** 400 000 Medicare Part-B claims (dummy PHI-stripped)  
**Goal:** Identify top denial reasons & trending revenue loss  

### Key Metrics
- Denied Claim Count & Revenue by Month  
- Top 10 Denial Reason Codes  
- YTD Denial Rate %  

### Publish Steps
1. Open `Medicare_Denials.pbix` → **File → Publish to Power BI**  
2. Check “Allow public sharing” → copy URL → shorten via bitly  
3. Add URL to root README
