### 📈 Revenue Cycle Performance Analysis

SQL | Excel | Healthcare Operations

Analyzed key revenue cycle metrics to evaluate billing efficiency, delays, and accounts receivable performance within a healthcare setting.

#### 🔍 Key Highlights

* Tracked DNFB (Discharged Not Final Billed) to identify billing delays
* Measured discharge-to-bill lag time to evaluate operational efficiency
* Analyzed A/R > 120 days to assess aging receivables and cash flow risk

#### 💡 Business Value

* Identifies bottlenecks in the billing process
* Highlights revenue leakage and delayed reimbursements
* Supports process improvements to accelerate cash flow

#### 🛠 Tools & Techniques

* SQL (data extraction and KPI calculations)
* Excel (Power Query, pivot tables, and reporting)
* Frequency : Daily refresh (Power Query → SQL Server) 

#### 🔄 How to Reproduce

1. Open Excel
2. Navigate to **Data → Get Data → From SQL Server**
3. Execute `revenue_cycle_kpi.sql`
4. Load data into the model
5. Build pivot tables and charts for KPI tracking
